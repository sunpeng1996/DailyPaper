    AutoScientists: Self-Organizing Agent Teams for Long-Running Scientific Experimentation

    AutoScientists: 自组织多智能体团队实现长周期科学实验自动化

    🧭 去中心化多智能体自组织协作

    Harvard University

    Shanghua Gao, Ada Fang, Marinka Zitnik

    提出去中心化自组织多智能体团队，通过共享状态与动态重组，在生物医学、GPT优化和蛋白质工程中超越现有科学代理

    Multi-AgentSelf-OrganizationScientific DiscoveryLong-HorizonDecentralizedParallel Experiments

    📌 工作参考价值

    • 去中心化团队协作范式可迁移至多 Agent 推荐系统优化：通过共享状态（实验日志、死胡同注册表、论坛）让 Agent 相互“看到”对方的尝试与失败，避免重复探索，适合电商中多目标排序、个性化策略的并行搜索。
    • 分析师与实验者角色分离：分析师 Agent 专职审核实验方向、生成和排序提案；实验者 Agent 实施并记录结果。这种分工可以用于生成式推荐中，由“策略脑”规划增量实验，“执行手”跑 A/B 测试。
    • 动态重组与死胡同注册：当某方向连续无提升时，自动触发讨论并重组团队，将探索资源转移到新方向。可用于 Agent 系统在在线学习场景中的自适应调整。
    • 实验前同行批评过滤：利用 Agent 间的异步讨论筛掉弱提案，节省算力。可借鉴到搜索链路中的策略预审或流量分配前的集体评估。

    动机： 现有 AI 科学代理在处理长期、多方向、动态变化的实验搜索时存在瓶颈：单代理系统沿单一轨迹优化，多代理系统依赖中央规划器或静态任务分解，难以在证据积累的过程中并行探索、记录失败并重定向。科学研究需要持续追踪竞争性假设、利用失败数据避免重蹈覆辙，这要求团队能够自发调整方向。

    方法关键点：

    • 去中心化自组织：无中央协调者，所有代理通过共享状态协作。
    • 共享状态四层结构：冠军模型、实验日志、公共论坛和团队本地状态（队列、死胡同注册表、假设文档）。
    • 双循环机制：讨论阶段（代理提议方向、互相批评、动态组成团队）与执行阶段（团队并行实验，分析师管理知识，实验者执行训练）。
    • 死胡同注册与跨团队知识共享：所有失败方向被记录，跨团队可见，减少冗余探索；当团队持续无改进时触发重组。
    • 实验前置过滤：代理在论坛中对提案进行批评，只有通过审查的实验才能执行。

    关键实验与结果：

    • BioML-Bench（24项生物医学ML任务）：平均领导板百分位74.40%，比Autoresearch的66.07%提升+8.33%，尤其在药物发现任务提升显著（46.16%→64.52%）。
    • GPT nanochat训练优化：达到相同验证损失比Autoresearch少用1.9倍实验数（34 vs 65）；从AutoScientists冠军出发继续搜索，发现7个有效改进（最终验证损失0.9730），而单代理无任何接受（0个改进）。
    • ProteinGym蛋白质适应性预测：在ACE2–Spike结合任务上，将Spearman相关系数从0.747提升至0.840（+12.5%）；冻结方法直接迁移至217个DMS检测，平均Spearman ρ从0.657提升至0.700。
    • 消融实验：移除分析师、取消跨代理反馈、固定团队组成或独立运行均导致性能显著下降，四个组件分别在不同任务中起关键作用。

    核心贡献一句话：将长期科学实验转化为去中心化智能团队的集体搜索过程，通过共享记忆、动态团队重组和实验前审查，让代理持续发现改进，而非在单个轨迹上停滞。
