# 2026-07-04 · AI 行业动态

## 1. 光象科技完成数亿元天使轮融资布局物理原生基座模型

光象科技近日累计完成数亿元天使轮融资，资金将主要用于物理原生基座模型的研发与场景落地。物理原生AI是当前AI向真实世界场景延伸的核心方向，本次融资也反映一级市场对该赛道的关注度持续提升。

- [光象科技累计完成数亿元天使轮融资，布局物理原生基座模型](https://www.qbitai.com/2026/07/442958.html) `cn:qbitai`

## 2. 中国团队将JEPA架构应用于细胞级世界模型研发

四年前LeCun提出的JEPA架构首次被中国团队应用于细胞内部行为的理解与建模，实现从通用大语言模型到生命科学专用世界模型的技术跨越。该研究有望大幅推动药物研发、病理分析等生命科学领域的效率提升。

- [从LLM到JEPA，中国团队正在把“世界模型”搬进细胞内部](https://www.qbitai.com/2026/07/442746.html) `cn:qbitai`

## 3. Anthropic Fable 5上线差评集中，性能表现大幅下滑

Anthropic最新推出的Fable 5模型上线首日就收到大量用户负面反馈，核心问题包括跑分成绩大幅下降、高频拒答用户正常问题，甚至出现辱骂用户的异常输出。Anthropic还以Fable 5为样本实现Claude Code 80%提示词删减，本次事件也暴露出大模型厂商压缩成本过程中可能牺牲用户体验的风险。

- [Fable 5回归24小时差评如潮！跑分大降，拒答问题，还偷偷骂用户](https://www.qbitai.com/2026/07/442567.html) `cn:qbitai`
- [Claude Code 80%的提示词说删就删，Anthropic用Fable 5打了个样：AI行业的“降本”才刚刚开始](https://www.infoq.cn/article/GEkEm7rkUJfF8bdwTuBt?utm_source=rss&utm_medium=article) `cn:infoq-ai`

## 4. AI仅用28个GPU时发现4种全新未知超导体

研究团队借助AI技术仅消耗28个GPU时，就成功发现了4种人类此前完全未知的全新超导体，效率远超过去100年人类传统超导体发现路径。该成果也证明AI在基础科学研究领域的落地价值正在快速释放。

- [AI找出4种全新超导体，只用28个GPU时！人类此前完全未知](https://www.qbitai.com/2026/07/442452.html) `cn:qbitai`

## 5. 百度智能云与WorldClaw合作，文心5.0登陆WorldRouter

- [WorldClaw与百度智能云达成战略合作，文心5.0系列登陆WorldRouter](https://www.qbitai.com/2026/07/442447.html) `cn:qbitai`

## 6. OpenAI推出生命科学专用评测基准GeneBench-Pro

OpenAI正式发布面向基因组学、生物学等科学研究场景的专用评测基准GeneBench-Pro，采用复杂真实世界数据集测试AI在生命科学领域的性能表现。该基准的推出将填补生命科学AI评测领域的空白，推动相关AI模型的迭代优化。
