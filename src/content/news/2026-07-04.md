# 2026-07-04 · AI 行业动态

## 1. OpenAI发布基因领域AI基准测试GeneBench-Pro

OpenAI推出面向基因组学、生物学及科研场景的AI性能基准测试GeneBench-Pro，采用复杂真实世界数据集。该基准可精准衡量大模型在生命科学领域的落地能力，填补了垂直领域评测标准的空白。

- [Inside Genebench-Pro](https://openai.com/index/genebench-pro/case-studies) `blog:openai-news`
- [Introducing GeneBench-Pro](https://openai.com/index/introducing-genebench-pro) `blog:openai-news`

## 2. OpenAI与惠普达成Frontier级战略合作伙伴关系

惠普升级与OpenAI的Frontier合作，双方将在客户体验优化、软件开发、企业运营等多个场景落地AI能力。本次合作也是OpenAI拓展To B商业化落地的重要布局。

- [HP Inc. launches Frontier strategic partnership with OpenAI](https://openai.com/index/hp-frontier-partnership) `blog:openai-news`

## 3. OpenAI拟向美国政府出让5%股份缓解监管压力

据多方消息，OpenAI正与美国特朗普政府进行早期谈判，计划出让5%股权以降低监管阻力。目前相关协商仍在推进中，该动作也为全球AI公司的监管合规提供了新参考。

- [OpenAI ‘in early talks to give 5% stake to US government’](https://www.theguardian.com/technology/2026/jul/02/openai-stake-us-government-ai-sam-altman) `hn · 133pts`
- [OpenAI proposes 5% stake to Trump administration to ease Washington pressure](https://www.cnbc.com/2026/07/02/openai-proposes-us-government-own-5percent-stake-to-address-political-blowback.html) `hn · 8pts`

## 4. Anthropic Fable 5发布后口碑暴跌引发行业讨论

Anthropic最新推出的Fable 5上线24小时就遭遇大量差评，出现跑分大幅下降、问题拒答甚至辱骂用户等问题。同时Claude Code也被曝出存在提示词随意删减的情况，引发行业对AI厂商降本牺牲产品体验的讨论。

- [Claude Code 80%的提示词说删就删，Anthropic用Fable 5打了个样：AI行业的“降本”才刚刚开始](https://www.infoq.cn/article/GEkEm7rkUJfF8bdwTuBt?utm_source=rss&utm_medium=article) `cn:infoq-ai`

## 5. 阿里因后门风险拟在内部禁用Claude Code

据消息人士透露，阿里巴巴计划在办公场景全面禁用Claude Code，原因是该产品被指控存在后门风险，企业数据安全隐患突出。本次事件也再次引发了企业对境外AI工具数据安全的关注。

- [Alibaba to ban Claude Code in workplace over alleged backdoor risks, source says](https://www.reuters.com/world/china/alibaba-ban-claude-code-workplace-over-alleged-backdoor-risks-source-says-2026-07-03/) `hn · 322pts`

## 6. 中国团队将JEPA架构应用于细胞内部研究

中国科研团队将四年前LeCun提出的JEPA世界模型架构应用于细胞内部机制研究，突破了传统生命科学研究的算力和模型瓶颈。该研究为世界模型在生物领域的落地开辟了新路径。

- [从LLM到JEPA，中国团队正在把“世界模型”搬进细胞内部](https://www.qbitai.com/2026/07/442746.html) `cn:qbitai`

## 7. 百度智能云与WorldClaw达成战略合作

WorldClaw与百度智能云正式达成战略合作，文心大模型5.0系列能力将接入WorldRouter平台。双方将共同探索AI多场景落地的可能性，进一步扩大文心大模型的应用边界。

- [WorldClaw与百度智能云达成战略合作，文心5.0系列登陆WorldRouter](https://www.qbitai.com/2026/07/442447.html) `cn:qbitai`

## 8. AI仅用28个GPU时发现4种全新超导体

科研团队借助AI仅花费28个GPU时就发现了4种人类此前完全未知的全新超导体，效率远超人类过去百年的超导体发现速度。该成果为材料科学领域的AI应用提供了标杆案例。

- [AI找出4种全新超导体，只用28个GPU时！人类此前完全未知](https://www.qbitai.com/2026/07/442452.html) `cn:qbitai`

## 9. 谷歌宣布7月17日关停Gemini Code Assist

谷歌宣布旗下代码辅助工具Gemini Code Assist将在7月17日正式停止服务。该产品的下线也反映出AI代码工具赛道的竞争日趋激烈，中小玩家生存空间被持续挤压。

- [Gemini Code Assist will be shut down on July 17](https://docs.cloud.google.com/gemini/docs/code-review/review-repo-code) `hn · 62pts`

## 10. Netflix推出端到端生成式首页方案GenPage

Netflix公开了其端到端生成式首页构建方案GenPage，该方案采用自回归方式逐行生成首页内容，每部分内容都基于已有内容的上下文生成。该方案可大幅提升内容推荐的个性化程度，优化用户体验。

- [GenPage: Towards End-to-End Generative Homepage Construction at Netflix](https://netflixtechblog.com/genpage-towards-end-to-end-generative-homepage-construction-at-netflix-77146fba8a08?source=rss----2615bd06b42e---4) `blog:netflix-tech`

## 11. Meta计划出售闲置AI算力布局云业务

Meta正在构建新的云业务板块，计划将自身闲置的AI算力对外出售。这也是Meta在AI基础设施领域变现的最新尝试，将进一步加剧全球云服务市场的竞争。

- [Meta building cloud business to sell excess AI capacity](https://www.reuters.com/business/meta-sell-excess-ai-computing-capacity-via-cloud-business-bloomberg-news-reports-2026-07-01/) `hn · 30pts`
