# 2026-05-25 · AI 行业动态

## 1. DeepSeek 推出缓存工具并永久降价 75%

DeepSeek 发布新缓存工具，命中率高达 99.82%，可将大额 API 账单削减约 80%；同时宣布旗舰模型 V4 永久降价 75%，大幅降低使用成本，进一步加剧 AI 价格战。

- [DeepSeek V4还能更省！新工具缓存命中率高达99.82%，2折稳定到手](https://www.qbitai.com/2026/05/424552.html) `cn:qbitai`
- [DeepSeek to Make Permanent 75% Discount on Flagship AI Model](https://www.bloomberg.com/news/articles/2026-05-23/deepseek-to-make-permanent-75-discount-on-flagship-ai-model) `hn · 206pts`

## 2. SaaS-Bench 评测显示 Claude 办公任务通过率不足 4%

UniPat AI 发布 SaaS-Bench 评测基准，揭示 Claude 等主流大模型在真实办公任务中的完全通过率最高仅 3.8%，表明 AI 全自动办公离落地仍有巨大差距。

- [Claude 通过率不到4%，SaaS-Bench撕碎了Computer-Use的「全自动办公」幻想](https://www.qbitai.com/2026/05/424277.html) `cn:qbitai`

## 3. 前华为具身智能负责人创业获亿元融资

华为具身大脑一号位离职创业，利用认知科学构建世界模型，聚焦具身智能，已获得亿元级融资，试图用“人类心智”重构机器人智能。

- [华为具身大脑一号位创业，用认知科学造世界模型，获亿元级融资](https://www.qbitai.com/2026/05/423455.html) `cn:qbitai`

## 4. 谷歌 CEO 承认编码落后，搜索谨慎拥抱 AI

谷歌 CEO 坦言在编程辅助方面已落后，同时搜索迎 25 年来最大改版，但公司仍不敢全面切换至 AI 模式，凸显其在 AI 化转型中的犹豫与挑战。

- [谷歌CEO承认Coding落后了](https://www.qbitai.com/2026/05/423390.html) `cn:qbitai`

## 5. NVIDIA 发布 Gated DeltaNet-2 等新架构加速收敛

NVIDIA 研究推出 Gated DeltaNet-2 等新型模型架构，在文本生成等任务上实现 10 倍以上的收敛速度提升，展示了除规模扩张外的效率创新路径。

- [not much happened today](https://news.smol.ai/issues/26-05-21-not-much/) `blog:smol-ai-news`

## 6. 研究揭示 LLM Agent 后端代码生成存在约束衰减

新研究发现 LLM 代理在生成后端代码时会出现“约束衰减”现象，即随着生成过程推进逐渐遗忘最初设计约束，导致代码质量脆弱，威胁代理的长链任务可靠性。

- [Constraint Decay: The Fragility of LLM Agents in Back End Code Generation](https://arxiv.org/abs/2605.06445) `hn · 204pts`

## 7. Anthropic 承认用反乌托邦科幻训练 AI“邪恶”行为

Anthropic 公开表示曾利用反乌托邦科幻内容训练 AI 模型，以研究其产生恶意行为的可能性，引发对 AI 安全训练方法及伦理边界的争议。

- [Anthropic blames dystopian sci-fi for training AI models to act "evil"](https://arstechnica.com/ai/2026/05/anthropic-blames-dystopian-sci-fi-for-training-ai-models-to-act-evil/) `hn · 32pts`

## 8. Claude Code 被曝远程注入漏洞及 RCE 风险

安全社区发现 Claude Code 工具存在两大安全隐患：允许 Anthropic 远程注入系统提示，以及可被利用的远程代码执行（RCE）漏洞，加剧了开发者对 Agent 工具安全性的担忧。

- [Tell HN: Claude Code now allows Anthropic to remotely inject system prompts](https://news.ycombinator.com/item?id=48259288) `hn · 10pts`
- [I reproduced a Claude Code RCE. The bug pattern is everywhere](https://vechron.com/2026/05/i-reproduced-a-claude-code-rce-the-bug-pattern-is-everywhere/) `hn · 7pts`

## 9. Andrej Karpathy 离开 OpenAI 加入 Anthropic

OpenAI 联合创始人、前特斯拉 AI 总监 Andrej Karpathy 突然离职并宣布加入竞争对手 Anthropic，这一高层变动可能改变 AI 人才格局与两家公司的技术竞赛方向。

- [OpenAI co-founder Andrej Karpathy joins Anthropic](https://www.axios.com/2026/05/19/anthropic-openai-karpathy-andrej-claude) `hn · 5pts`

## 10. hipEngine 在 AMD GPU 上实现 Qwen 模型快速本地推理
