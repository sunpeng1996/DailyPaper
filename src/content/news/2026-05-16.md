# 2026-05-16 · AI 行业动态

## 1. OpenAI 将 Codex 集成到 ChatGPT 移动应用

OpenAI 正式在 ChatGPT 移动端上线 Codex，用户可直接在手机上进行代码生成与交互。该功能此前仅在网页版提供，移动端接入后进一步降低了 AI 编程的门槛，也加剧了与 GitHub Copilot 等移动编码工具的竞争。

- [Codex is now in the ChatGPT mobile app](https://openai.com/index/work-with-codex-from-anywhere/) `hn · 469pts`
- [Codex is now available on mobile via ChatGPT app](https://twitter.com/openai/status/2055016850849993072) `hn · 38pts`
- [OpenAI's Codex is now in the ChatGPT mobile app](https://www.theverge.com/ai-artificial-intelligence/930763/openai-codex-chatgpt-ios-android-app-preview) `hn · 12pts`

## 2. OpenAI 通过 Plaid 将 ChatGPT 连接到银行账户

OpenAI 宣布与金融数据平台 Plaid 合作，允许 ChatGPT 用户授权访问银行账户信息，从而获得个性化的财务建议和分析。此举标志着 OpenAI 开始深入金融服务领域，但也引发了隐私和数据安全的担忧。

- [OpenAI is connecting ChatGPT to bank accounts via Plaid](https://firethering.com/chatgpt-bank-account-plaid-openai/) `hn · 72pts`

## 3. Anthropic 推出法律垂直产品 Claude for Legal

Anthropic 发布专为法律行业设计的 Claude for Legal，提供合同审查、法律研究、文书起草等功能。该产品基于 Claude 模型微调，瞄准律所和企业法务部门，是 Anthropic 在垂直行业商业化的关键一步。

- [Claude for Legal](https://github.com/anthropics/claude-for-legal) `hn · 197pts`

## 4. Anthropic 与盖茨基金会达成 2 亿美元合作

Anthropic 与比尔及梅琳达·盖茨基金会建立战略合作伙伴关系，后者将投资 2 亿美元，共同利用 AI 促进全球健康、教育和扶贫事业。此次合作凸显了 Anthropic 在非营利领域的布局，也为其带来稳定的长期资金与场景验证。

- [Anthropic forms $200M partnership with the Gates Foundation](https://www.anthropic.com/news/gates-foundation-partnership) `hn · 121pts`

## 5. Anthropic 调整订阅计划，移除 Agent SDK 等权益

Anthropic 更新 Claude 订阅条款，原包含的 Agent SDK 和 `claude -p` 命令行工具不再属于订阅计划，可能转为单独收费或更高阶套餐。该变动引发部分开发者不满，认为削弱了订阅价值，同时也反映出 Anthropic 对 API 货币化策略的调整。

- [Claude subscriptions no longer include Agent SDK and Claude -p usage](https://www.xda-developers.com/anthropics-claude-subscriptions-no-longer-include-agent-sdk-and-claude-p-usage/) `hn · 17pts`
- [Anthropic moves Claude Code SDK and claude -p out of subscription plans](https://twitter.com/ClaudeDevs/status/2054610152817619388) `hn · 9pts`

## 6. DeepSeek 发布新一代开源模型 DeepSeek V4

深度求索发布 DeepSeek V4 大模型，并称其为“前沿实验室曾畏惧的开源模型”，在多项基准上达到或超越领先闭源模型。该模型继续坚持开源，进一步巩固了 DeepSeek 在开源社区的影响力，并对 OpenAI、Google 等形成竞争压力。

- [DeepSeek V4: The Open-Source Model Frontier Labs Feared](https://helloai.com/articles/deepseek-v4-open-source-frontier-parity) `hn · 56pts`

## 7. 蚂蚁开源百灵 Ring-2.6-1T，提升 Agent 执行能力

蚂蚁集团开源百灵 Ring-2.6-1T 模型，专门增强 Agent 任务执行能力，在 AIME 26 等基准上得分达 95.83。该模型聚焦智能体场景，可处理复杂多步推理和工具调用，为开源 Agent 框架提供了新的基座选择。

- [蚂蚁百灵 Ring-2.6-1T 开源 Agent 执行能力全面增强](https://www.qbitai.com/2026/05/417961.html) `cn:qbitai`

## 8. 阿里发布 AI 编程工具 Qoder 1.0

阿里巴巴推出 Qoder 1.0，一个可接管代码生成、验证和全流程交付的 AI 编程助手，支持 Windows、macOS 和 Linux 系统。该工具面向开发者，旨在从需求理解到代码交付实现端到端自动化，进一步降低了软件开发的门槛。

- [Need is all you need：AI接手Coding后，程序员最值钱的能力只剩这一项?](https://www.qbitai.com/2026/05/418035.html) `cn:qbitai`
- [阿里发布Qoder 1.0，可全面接管代码生成、验证和交付流程](https://www.qbitai.com/2026/05/418027.html) `cn:qbitai`

## 9. Meta 开始收集员工键鼠操作训练 AI

Meta 被曝将收集员工鼠标移动轨迹和键盘输入数据，用于训练内部 AI 模型。该政策引发隐私争议和员工不安，也反映出科技巨头在数据获取上的激进策略，可能触及劳动与隐私法规边界。

- [Meta to Start Capturing Employee Mouse Movements, Keystrokes for AI Training](https://daringfireball.net/linked/2026/05/10/meta-employee-spyware) `hn · 17pts`

## 10. Google DeepMind 员工投票组建工会，抗议军事 AI 项目

Google DeepMind 员工以多数票通过决议，决定组建工会，直接起因是反对公司将 AI 技术用于军事合同。这一行动反映了 AI 从业者日益增长的伦理关切，也可能会对 Google 后续的国防项目产生人事和政策影响。

- [Google DeepMind Workers Vote to Unionize over Military AI Deals](https://www.wired.com/story/google-deepmind-workers-vote-to-unionize-over-military-ai-deals/) `hn · 15pts`
