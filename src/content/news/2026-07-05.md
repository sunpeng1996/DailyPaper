## 1. 光象科技完成数亿元天使轮融资布局物理原生基座模型

光象科技近期累计完成数亿元天使轮融资，资金将重点投入物理原生基座模型的技术研发与场景落地。物理原生基座模型以真实世界物理规律为建模核心，是AI基础模型领域的新兴研发赛道。

## 2. 国内团队将JEPA架构应用于细胞级世界模型研究

国内科研团队首次将LeCun提出的JEPA架构落地于细胞层面的世界模型研发，尝试通过AI模拟细胞内部的复杂生理活动。该研究拓展了JEPA架构的应用边界，为生命科学领域的AI应用提供了新的技术路径。

- [从LLM到JEPA，中国团队正在把“世界模型”搬进细胞内部](https://www.qbitai.com/2026/07/442746.html) `cn:qbitai`

## 4. 阿里巴巴因后门风险将禁用Claude Code

据消息人士透露，阿里巴巴将在内部工作场所全面禁用Claude Code，核心原因是其存在潜在的后门安全风险。该举措也反映出企业对海外AI工具的数据安全问题关注度持续提升。

- [Alibaba to ban Claude Code in workplace over alleged backdoor risks, source says](https://www.reuters.com/world/china/alibaba-ban-claude-code-workplace-over-alleged-backdoor-risks-source-says-2026-07-03/) `hn · 331pts`

## 5. OpenAI发布基因领域基准测试集GeneBench-Pro

OpenAI正式推出面向基因组学、生物学等科研场景的AI性能基准测试集GeneBench-Pro，采用真实世界复杂数据集作为测试用例。该基准将为生命科学领域AI模型的性能评估提供统一的衡量标准。

- [Inside Genebench-Pro](https://openai.com/index/genebench-pro/case-studies) `blog:openai-news`

## 6. Meta称即将推出的大模型已追平OpenAI旗舰款

Meta AI负责人公开表示，其即将发布的全新大模型性能已经追平OpenAI的旗舰级产品。该消息意味着大模型头部市场的竞争格局可能迎来新的变化。

- [Meta AI chief says their coming LLM has caught up with OpenAI's flagship model](https://www.businessinsider.com/meta-ai-model-catches-up-openai-gpt-5-says-2026-7) `hn · 14pts`

## 7. OpenAI与惠普达成Frontier战略合作伙伴关系

惠普宣布扩大与OpenAI的Frontier战略合作范围，双方将在用户体验优化、软件开发、企业级运营等多个场景落地AI能力。该合作也是OpenAI企业服务商业化落地的重要进展。

- [HP Inc. launches Frontier strategic partnership with OpenAI](https://openai.com/index/hp-frontier-partnership) `blog:openai-news`

## 8. 达摩院发布可发现新型超导体的AI智能体

阿里达摩院推出的AI智能体仅用28个GPU时就发现了4种人类此前未知的全新超导体，研发效率远超过去100年的人类研究速度。该成果验证了AI在基础科学研究领域的巨大应用潜力。

- [AI找出4种全新超导体，只用28个GPU时！人类此前完全未知](https://www.qbitai.com/2026/07/442452.html) `cn:qbitai`
- [Damo Academy unveils an AI agent able to discover superconductors](https://www.scmp.com/tech/big-tech/article/3359335/alibabas-elements-claw-ai-agent-unearths-four-new-superconductors) `hn · 7pts`

## 9. Netflix推出端到端生成式主页构建工具GenPage

Netflix公开了其研发的端到端生成式主页构建工具GenPage，可通过自回归方式逐行生成个性化主页内容。该工具将大幅提升Netflix主页的个性化推荐效率与用户匹配度。

- [GenPage: Towards End-to-End Generative Homepage Construction at Netflix](https://netflixtechblog.com/genpage-towards-end-to-end-generative-homepage-construction-at-netflix-77146fba8a08?source=rss----2615bd06b42e---4) `blog:netflix-tech`

## 10. Google宣布7月17日关闭Gemini Code Assist

Google官方宣布将于2025年7月17日正式停止Gemini Code Assist服务。该服务是Google面向开发者推出的AI编码辅助工具，此次停运也反映出AI编码工具赛道的竞争日趋激烈。

- [Gemini Code Assist will be shut down on July 17](https://docs.cloud.google.com/gemini/docs/code-review/review-repo-code) `hn · 62pts`

## 11. OpenAI发布欧盟AI劳动力影响研究报告

OpenAI发布针对欧盟市场的AI劳动力影响研究报告，梳理了AI技术对不同职业的自动化替代、岗位增长、工作流调整等多维度影响。该报告将为欧盟的AI就业政策制定提供参考依据。

- [Mapping Europe’s AI Workforce Opportunity](https://openai.com/index/mapping-ai-jobs-transition-eu) `blog:openai-news`
