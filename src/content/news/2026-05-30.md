# 2026-05-30 · AI 行业动态

## 1. Anthropic 发布 Claude Opus 4.8

Anthropic 推出 Opus 4.8，在协作和编码行为上有所提升，但文档解析等部分基准出现倒退，整体提升有限。该模型已通过 API 和 Claude Code 提供。

- [not much happened today](https://news.smol.ai/issues/26-05-29-not-much/) `blog:smol-ai-news`
- [Claude Opus 4.8](https://www.anthropic.com/news/claude-opus-4-8) `hn · 1731pts`

## 2. Anthropic 完成 650 亿美元 H 轮融资

Anthropic 以 9650 亿美元估值完成 650 亿美元 H 轮融资，由 Altimeter、Dragoneer 等领投，年化营收超 470 亿美元，AI 赛道投资热度再创新高。

- [Anthropic raises $65B in Series H at a $965B post-money valuation, releases Opus 4.8 and Dynamic Workflows](https://news.smol.ai/issues/26-05-28-anthropic-series-h/) `blog:smol-ai-news`
- [Anthropic raises $65B in Series H funding at $965B post-money valuation](https://www.anthropic.com/news/series-h) `hn · 359pts`

## 3. Anthropic 为 Claude Code 推出动态工作流

Claude Code 新增 Dynamic Workflows 功能，支持更灵活的代理式编码流程，进一步强化 AI 开发工具能力。

- [Dynamic Workflows in Claude Code](https://claude.com/blog/introducing-dynamic-workflows-in-claude-code) `hn · 168pts`
- [Anthropic raises $65B in Series H at a $965B post-money valuation, releases Opus 4.8 and Dynamic Workflows](https://news.smol.ai/issues/26-05-28-anthropic-series-h/) `blog:smol-ai-news`

## 4. OpenAI 宣布生物防御项目 Rosalind

OpenAI 公布 Rosalind Biodefense 计划，将 AI 技术拓展至生物安全领域，显示其在垂直行业的布局。

- [OpenAI Announces Rosalind Biodefense](https://openai.com/index/strengthening-societal-resilience-with-rosalind-biodefense/) `hn · 18pts`

## 5. 腾讯发布 AI 游戏创作平台

腾讯推出 AI 游戏创作平台，零基础用户也可制作复杂游戏，推动游戏开发民主化，被视为下一波 AI 创作浪潮。

- [下限零基础，上限肝大作！腾讯这个AI游戏创作平台，太野了](https://www.qbitai.com/2026/05/426447.html) `cn:qbitai`

## 6. 腾讯发布智能体创意工作室 Miora

腾讯发布 Miora，一个让单人拥有完整创意工作室的智能体平台，聚焦设计领域效率提升。

- [创意设计版WorkBuddy来了！腾讯发布智能体创意工作室Miora](https://www.qbitai.com/2026/05/426400.html) `cn:qbitai`

## 7. DeepSeek API 推理成本大幅下降

DeepSeek 通过优化将 API 价格压至极低，有用户从 Claude 切换后成本减少 99%，对高价模型形成冲击。

- [I cut my AI API costs 99% by switching from Claude to DeepSeek](https://twitter.com/kevinsays/status/2059724146221662339) `hn · 26pts`
- [DeepSeek Slashes AI Costs to Cents](https://businessanalytics.substack.com/p/deepseek-slashes-ai-costs-to-cents) `hn · 13pts`

## 8. 面壁智能启动端侧 AI 开源周

面壁智能通过开源周系统性释放端侧 AI 组件，旨在定义端侧 AI 终局，展示从硬件到算法的全栈能力。

- [面壁智能「开源周」：一场定义端侧 AI 终局的系统性「亮剑」](https://www.qbitai.com/2026/05/426542.html) `cn:qbitai`

## 9. ITBench-AA 发布，前沿模型企业 IT 代理任务得分不足 50%

Artificial Analysis 与 IBM 联合推出 ITBench-AA，这是首个面向代理式企业 IT 任务的基准，前沿 LLM 得分均低于 50%，凸显自动化运维落地的差距。

- [ITBench-AA: Frontier Models Score Below 50% on the First Benchmark for Agentic Enterprise IT Tasks — by Artificial Analysis and IBM](https://huggingface.co/blog/ibm-research/itbench-aa) `blog:hf-blog`

## 10. 神秘公司单月意外花费 5 亿美元在 Claude 上
