# 2026-05-23 · AI 行业动态

## 1. DeepSeek 即将推出 Code 模型，崔添翼挂帅

DeepSeek 正筹备发布代码生成模型，由 ACM 金牌得主崔添翼领导，同时公司近期获得 700 亿融资，加速模型矩阵扩张。

- [融资700亿！DeepSeek Code真要来了，ACM金牌大神崔添翼挂帅](https://www.qbitai.com/2026/05/422624.html) `cn:qbitai`

## 2. DeepSeek V4 Pro API 永久降价至原先 1/4

DeepSeek 宣布 V4 Pro 模型 API 在 75 折促销结束后将正式调降为原价的四分之一，大幅降低使用门槛，加剧大模型价格竞争。

- [DeepSeek makes the V4 Pro price discount permanent](https://api-docs.deepseek.com/quick_start/pricing) `hn · 340pts`

## 3. 智谱 GLM 代码生成速度达 400 tokens/s

智谱展示其模型在代码生成任务上达到 400 tokens/s 的极高推理速度，被称为“喷代码”，在同类产品中表现突出。

- [顶流里最快！智谱，你是在「喷」代码吧](https://www.qbitai.com/2026/05/422511.html) `cn:qbitai`

## 4. 腾讯混元开源翻译模型 Hy-MT2 并上线小程序

腾讯发布并开源了新一代翻译模型 Hy-MT2，重点提升指令遵循能力，同时推出“腾讯 Hy 翻译”小程序，降低使用门槛。

- [腾讯混元开源全新翻译模型Hy-MT2 ，上线小程序「腾讯Hy翻译」](https://www.qbitai.com/2026/05/422068.html) `cn:qbitai`

## 5. Google I/O 发布 Gemini 3.5 Flash、Omni 及 Agent Stack

Google 在 I/O 大会上重新定位 Gemini 为消费与开发者平台，发布三款核心产品：快速代理与编码任务的 Gemini 3.5 Flash、多模态 Gemini Omni 以及面向代理的全栈工具。

- [Google I/O 2026: Gemini 3.5 Flash, Omni, and Google’s Agent Stack](https://news.smol.ai/issues/26-05-19-not-much/) `blog:smol-ai-news`

## 6. Anthropic 推出 MCP 隧道，支持私有代理访问内部系统

Anthropic 推出 MCP 隧道功能，允许企业内部的 AI 代理安全地访问受保护的内部系统，扩展了代理的应用边界。

- [Anthropic 推出 MCP 隧道，供私有代理访问内部系统](https://www.infoq.cn/article/jvoDNDaa2bRzwrHQy7lT?utm_source=rss&utm_medium=article) `cn:infoq-ai`

## 7. 李飞飞团队发布空间智能评测基准

李飞飞团队发布了一个专门用于评测具身空间智能的新基准，被业界称为空间智能的 ImageNet，旨在推动具身模型标准化评估。

- [李飞飞再出手，空间智能的ImageNet来了](https://www.qbitai.com/2026/05/422738.html) `cn:qbitai`

## 8. 微软取消 Claude Code 许可证，推动自有工具

微软开始取消提供给内部的 Claude Code 许可证，被解读为转向推广自家的 AI 编程工具，加剧与 Anthropic 的竞争。

- [Microsoft starts canceling Claude Code licenses](https://www.theverge.com/tech/930447/microsoft-claude-code-discontinued-notepad) `hn · 165pts`

## 9. YC 创业公司 Runtime 推出沙盒编码代理

Runtime (YC P26) 发布沙盒化编码代理基础设施，让非工程师团队成员也能安全地使用 AI 代理执行代码，降低代理使用门槛。

- [Launch HN: Runtime (YC P26) – Sandboxed coding agents for everyone on a team](https://www.runtm.com/) `hn · 100pts`
