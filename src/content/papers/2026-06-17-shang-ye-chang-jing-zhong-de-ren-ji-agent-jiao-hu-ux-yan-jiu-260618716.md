**动机**：随着 AI Agent 被广泛集成到核心业务（如自动化客服、流程助手），人与 Agent 的交互直接影响价值创造，但缺少面向商业场景的 UX 原则与度量标准。本研究旨在填补这一空白，识别并评估能提升用户体验的设计原则，以促进用户采纳和信任。

**方法**：采用混合方法（mixed-methods），第一阶段通过定性研究（访谈/观察）探索人与 AI Agent 的交互模式，提炼用户需求与期望；第二阶段基于定性发现开发调查实验（survey experiment），定量评估特定设计元素（如解释性、可控性）的效果。整体由 SAP 研究团队主导，结合工业界与学术界视角。

**关键结果**：论文属于探索性基础研究，主要输出为一套面向商业 Agent 的 UX 原则、度量标准及评估方法；尚未报告大规模实验的量化结果，但为后续的直观化、高效交互设计提供了方法论和理论支撑。
