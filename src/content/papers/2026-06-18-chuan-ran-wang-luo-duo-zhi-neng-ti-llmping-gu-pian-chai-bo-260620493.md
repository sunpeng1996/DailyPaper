**动机**  
多智能体LLM系统中，智能体互相评估形成闭环反馈。若评估者存在系统偏好，这种偏差会通过评估-更新环路在多跳间传播，导致所有智能体策略趋同，丧失认知多样性（策略坍塌）。现有工作仅关注单评估者或成对效应，未分析多跳传播动力学。本文借鉴传染病建模思想，提出传染网络（Contagion Networks）框架，形式化量化并抑制评估偏差在多智能体网络中的传播。  

**方法关键点**  
- **交叉传染矩阵 Γ_N**：定义 N 智能体的成对传染系数 γ_{j→i}，衡量评估者 j 对智能体 i 策略权重分布的改变幅度。矩阵 Γ_N 的谱半径 ρ 决定传播动力学。  
- **适应机制 TTRL**：智能体通过 Test-Time Reinforcement Learning 乘性更新策略权重，不需修改 LLM 参数，轻量且兼容 API 调用。  
- **三个传播区**：由 ρ(Γ_N) 分类——ρ<1 抑制（偏差逐跳衰减），ρ≈1 持续，ρ>1 级联（全网偏好坍塌）。链拓扑下级联条件进一步简化为任意单跳 γ>1。  
- **多样性抑制定理**：当 k 个评估者偏好相似度 ≤τ，有效传染 γ_eff 随 k 以 1/√k 衰减；k≥3 即可将强传染压制到级联阈值以下。  

**关键实验与结果**  
使用 DeepSeek-chat 的三个变体（结构偏向/平衡/证据偏向）在 50 个任务（代码、数学、总结等）上进行 840 次 API 调用实验。  
- 成对传染矩阵所有 γ∈[0.143,0.304]，远低于跨模型值（0.85–1.3），系统处于抑制区；但全连接拓扑的谱半径 ρ(Γ₃)=1.402，暗示密集连接可能触发级联。  
- 3 跳链传播累积衰减因子 β₃=0.0055，偏差近乎完全消失。  
- 委员会规模从 k=1 增至 k=3，有效传染降低 72.4%（0.264→0.073），策略熵趋近最大熵 ln5≈1.609，证明多样性有效抑制传播。  

**核心结论**  
同模型评估者池天然抑制偏差传播，跨模型多样性反而放大传染；至少三个评估者组成的委员会能大幅削弱传染风险。部署多智能体系统前应测量 Γ 矩阵，监控策略熵，以防策略多样性悄然丧失。
