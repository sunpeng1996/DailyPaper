The agent engineering platform for building agents and LLM-powered applications.

**借鉴价值**：

- 丰富的工具集成和模型抽象，快速实现生成式推荐逻辑

## 2. [langgenius/dify](https://github.com/langgenius/dify) ★145013

**借鉴价值**：
- 可视化工作流设计，降低电商多代理系统开发门槛
- 内置 RAG 和记忆管理，便于构建个性化推荐助手
- 企业级部署能力，可扩展为电商客服和推荐引擎

## 3. [bytedance/deer-flow](https://github.com/bytedance/deer-flow) ★71069

**借鉴价值**：
- 长时任务编排，适合电商复杂流程（如全自动选品、竞品分析）
- 子代理架构可拆解电商多智体任务（市场调研、内容生成、策略优化）
- 记忆和沙盒机制保障生产环境稳定和安全，可直接复用架构

- 代理雇佣和报告机制启发 Agent 多智体优化中的职责分工与绩效监控

## 5. [vllm-project/vllm](https://github.com/vllm-project/vllm) ★82731

High-throughput and memory-efficient LLM inference and serving engine.

**能力**：采用 PagedAttention 实现高效注意力内存管理，支持连续批处理、量化、推测解码等，提供高吞吐低延迟的 LLM 服务。

**借鉴价值**：
- 为生成式推荐模型提供高性能推理，降低延迟和成本
- 支持高并发请求，适合电商推荐场景的大规模服务
- 量化功能可在资源有限环境下部署推荐模型

## 6. [huggingface/transformers](https://github.com/huggingface/transformers) ★161552

Framework for state-of-the-art machine learning models in text, vision, audio, and multimodal.

**能力**：提供数千个预训练模型（文本、图像、视频）的统一接口，支持训练和推理，可与 PyTorch/TensorFlow 互操作，涵盖推荐相关模型如 BERT、GPT、ViT 等。

**借鉴价值**：
- 直接用于生成式推荐模型的训练和微调（如商品描述生成、用户行为预测）
- 多模态模型支持，可融合图文信息用于电商内容推荐
- 活跃的生态和工具链，降低从研究到生产的门槛

## 7. [hiyouga/LlamaFactory](https://github.com/hiyouga/LlamaFactory) ★72126

Unified efficient fine-tuning framework for 100+ LLMs & VLMs.

**能力**：提供低资源微调方法（LoRA、QLoRA、全参数等），支持 100 多种大语言模型和视觉语言模型，集成训练、评估和推理，简化模型定制流程。

**借鉴价值**：
- 低成本微调电商领域的专用 LLM（如商品推荐对话、文案生成）
- 支持 VLMs，可用于基于图像的推荐（如以图搜图）
- 统一工作流节省工程时间，快速迭代生成式推荐模型

## 8. [firecrawl/firecrawl](https://github.com/firecrawl/firecrawl) ★132058

API to search, scrape, and turn web content into clean Markdown/structured data at scale.

**能力**：提供网页抓取、搜索和内容提取 API，处理 JS 渲染页面，输出结构化数据或 Markdown，适合大规模电商网站数据采集。

**借鉴价值**：

- 结构化输出可直接用于生成式推荐模型的训练数据构建
