## 1. [deepset-ai/haystack](https://github.com/deepset-ai/haystack) ★25614

生产级 AI 编排框架，构建 LLM 驱动的 RAG、Agent 和语义搜索管道。

**能力**：Haystack 提供模块化管道架构，支持检索增强生成（RAG）、Agent 工作流、路由、记忆、多模态等组件。基于 Python 生态，集成主流向量数据库（如 Elasticsearch、Pinecone）、LLM 和 Embedding 模型，可构建自定义的搜索、问答和对话系统，适合生产环境部署。

**借鉴价值**：
- 可直接复用 Haystack 的 RAG 管道设计模式，快速搭建电商商品搜索、知识库问答等检索增强系统。
- 借鉴其 Agent 工作流编排与 Tool/Retriever 抽象，实现多步推理的推荐 Agent（如从用户意图识别到商品召回+排序）。
- 参考其生产化监控、缓存、序列化等组件，提升自研搜索推荐系统的工程可靠性。

## 2. [labring/FastGPT](https://github.com/labring/FastGPT) ★28554

开箱即用的 AI 知识库平台，通过可视化 RAG 检索与工作流编排构建复杂问答系统。

**借鉴价值**：
- 参考其可视化工作流设计，为搜索推荐系统的多阶段处理（如查询理解→召回→精排→解释）提供低代码编排思路。
- 借鉴其知识库管理机制（文档分割、元数据过滤、混合检索），用于构建电商知识库或商品文档的问答模块。
- 利用其 LLM+检索的对话架构，快速实现对话式推荐或导购 Agent 的原型验证。

## 3. [agentscope-ai/agentscope](https://github.com/agentscope-ai/agentscope) ★27011

生产就绪的 Agent 框架，提供事件驱动、人机协同和多 Agent 编排能力。

**能力**：AgentScope 2.0 基于 Python，提供事件总线、记忆管理、多模态支持和工具集成等核心抽象。它支持单 Agent 及多 Agent 协作，内置可视化调试面板和分支推理，可轻松接入外部 API、数据库和 LLM 服务，适合构建复杂的自主 Agent 应用。

**借鉴价值**：
- 借鉴其事件驱动的 Agent 通信机制，设计推荐场景中多个 Agent（用户建模、召回、排序、解释）的松耦合协作。
- 参考内置的记忆管理抽象，为搜索推荐系统增加长短期用户记忆（如近期行为、长期偏好）的统一访问层。
- 利用其人机协同接口，在关键推荐节点引入人工干预（如冷启动、高风险决策），提升系统可控性。

## 4. [volcengine/OpenViking](https://github.com/volcengine/OpenViking) ★25832

专为 AI Agent 设计的上下文数据库，提供文件系统式的记忆、资源和技能管理。

**能力**：OpenViking 通过文件系统范式统一管理 Agent 所需的上下文（记忆、资源、技能），支持层次化上下文传递和自我进化。它提供高效存储、检索和版本控制，可无缝集成到 Agent 工作流中，解决大模型上下文窗口有限和记忆隔离问题。

**借鉴价值**：
- 复用其层次化记忆模型，为搜索推荐 Agent 构建用户短期画像（session 级）和长期偏好（profile 级）的差异化存储与检索策略。
- 参考技能即文件的理念，将推荐策略（如“新品推荐”“折扣推荐”）封装为可插拔的 Skills，便于 Agent 动态选择组合。
- 借鉴其上下文传递机制，在推荐多轮对话中保持连贯的意图追踪和结果解释。
