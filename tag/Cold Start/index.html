      Quantizing Intent: Cross-Domain Semantic IDs from Organic Activity for Industrial Ranking

      量化意图：基于有机活动的跨域语义ID用于工业广告排序

      🧭 跨域语义ID · 行为活动丰富度 ★ 深度解读

      LinkedIn

      提出跨域用户语义ID和行为活动丰富度原则，利用有机Feed信号提升广告冷启动CTR达+1.522%，同时RQ-FSQ匹配密集AUC且存储缩小30～280倍

      📌 工作参考价值（4 点）
      • 行为活动丰富度选源原则：跨域特征优先选择编码近期行为信号（如浏览、互动序列）的源，而非静态画像文本。实验显示 Feed Activity 直接聚合行为比纯文本嵌入多 +0.177% AUC，可作为电商/广告系统选择外部特征源的直接指导。
      • RQ-FSQ 离散化通用方案：将预训练用户/商品嵌入离散化时，可结合残差 VQ 保留全局结构与单维标量量化保留细节，匹配甚至超过原始密集嵌入 AUC。这一方法适用于压缩 LLM 编码器输出，在存储受限的推荐服务中能直接复用，实现 30～280 倍压缩而不掉点。
      • HDE 模块即插即用：使用前缀 n-gram 哈希表对任意 K 级离散 ID 序列编码，支持多源独立查表与求和，插入现有 CTR 特征层无需修改骨干网络。该设计可直接迁移到电商排序模型的特征工程环节，降低多源离散特征集成成本。
      • 冷启动行为桥接：将非广告域（如内容浏览、社交互动）的丰富行为量化为语义 ID，作为排序模型的用户侧特征，可有效缓解新用户/稀疏用户信号不足，冷启动段 CTR 提升达 +1.522%，对电商 Push、新客推荐场景有直接借鉴意义。

      Bridging the Cold-Start Gap: LLM-Powered Synthetic Data Generation for Natural Language Search at Airbnb

      用 LLM 合成查询与标签破解自然语言搜索冷启动

      🧭 自然语言搜索 · 合成数据冷启动 ★ 深度解读

      Airbnb

      用少量种子查询引导 LLM 生成带标咨询数据和标签，使查询分布接近真实用户，支持检索与排序模型冷启动训练

      📌 工作参考价值（4 点）
      • 用种子查询控制 LLM 生成风格：直接让 LLM 无约束生成会产出冗长、多属性的查询，与真实用户短促、不规范的表达严重不匹配。从用户调研或早期日志中收集数百条种子查询作为模板，可以大幅降低 KL 散度（本文从 4.95→0.66），实现更逼真的查询分布。在电商搜索、对话式推荐中可复用此思路，用小样本种子让合成数据贴合真实语言。
      • 三类提示模板平衡真实性与多样性：seed_controlled 严格保持种子结构，适合属性覆盖；seed_freeform 模仿风格但允许变化；variety 放松约束、提高温度，探索长尾意图。实际生成时按比例混合三种模板，既能保证主流查询的真实感，又能覆盖边缘场景。推荐系统在生成候选集时可借鉴这种分阶段控制的提示策略。
      • 困难样本构建的分层采样与提示技巧：为避免全是简单样本（所有模型都能答对），通过限制样本对相似度（同类别、嵌入距离过滤）和提示中加入“难度桶”、反事实编辑等手段生成有区分度的训练数据。这对于提高 ranking 模型的迭代信号非常重要，电商场景下可借鉴硬负样本挖掘来提升模型判别力。
      • 从 LLM 自生成数据中避开自偏好陷阱：用 LLM 评估 LLM 生成的内容容易出现系统性偏爱（本文 GPT 评估自身生成的对准确率高达 99%），实际训练中建议用构建式标签（对比生成）保证高保真，用 LLM 评估仅作离线质量检查；且通过属性锚定、人工校准、使用异构模型降低自我偏好风险。这对任何用 LLM 生成训练数据的团队都是重要提醒。

      Language-Induced Priors for Domain Adaptation

      语言诱导先验的冷启动域适应框架

      🧭 利用LLM文本先验指导冷启动域适应

      用LLM将目标域文本描述转化为源选择先验，嵌入EM算法解决冷启动负迁移

      📌 工作参考价值（4 点）
      • 电商推荐场景冷启动物品/用户：若有文本描述（标题、属性），可用类似LIP的方法通过LLM生成对相似源域（如成熟品类）的偏好概率，减少迁移时的负迁移风险。
      • 多智体协作中的经验复用：当任务用自然语言描述时，可借助LLM先验快速筛选相关历史任务数据，提升少样本策略学习效率。
      • 与现有预估模型兼容：框架仅要求模型有似然函数，可无缝集成到CTR/CVR预估的线性或深度模型，无需改动结构。
      • 工程实现轻量：先验可离线通过LLM计算文本嵌入相似度得到，在线阶段只是简单的概率加权，延迟可控。
