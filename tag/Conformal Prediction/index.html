      Learning What to Say to Your VLA: Mostly Harmless Vision Language Action Model Steering

      学会对VLA说什么：几乎无害的视觉语言动作模型引导

      🧭 具身智能 · VLA语言引导与安全拒绝 ★ 深度解读

      Carnegie Mellon University

      交互式搜索语言序列并利用共形预测实现无害化语言引导，显著提升冻结VLA的任务成功率

      📌 工作参考价值（4 点）
      • 冻结模型的语言层干预：在推荐或对话Agent中，若已有强大的基座模型（如LLM），无需微调模型权重，通过学习一个轻量的语言策略（LFP）在测试时动态改写用户指令或系统提示，即可引导模型行为，避免重训成本。
      • 结构化语言空间搜索与蒸馏：通过VLM生成叙述先验，再基于LLM产生局部语义扰动，最后用闭环rollout筛选高奖励序列并蒸馏到小模型，这种方法可直接用于生成式推荐中的query/文案优化：先由大模型提供候选，再通过在线A/B或模拟环境筛选，最后蒸馏到轻量策略。
      • 共形预测实现安全拒绝：训练一个改进头预测干预收益，再用共形预测校准阈值，当分布外时拒绝语言引导，退回原始指令。这种“无害优先”的安全机制可以应用到风险敏感的电商Agent中，如仅在高置信度下进行动态价格推荐或主动营销。
      • 闭环语言反馈的恢复行为：与一次性改写指令不同，闭环语言策略能根据环境状态实时调整语言，实现错误恢复。在对话式推荐或多步Agent协作中，实时反馈和状态适应是提升任务完成率的关键。

      Conformal Risk-Averse Decision Making with Action Conditional Guarantee

      带有动作条件保证的共形风险规避决策

      🧭 共形预测与风险规避决策

      引入动作条件共形预测，为每项决策动作提供显式安全保证，提升风险规避决策可靠性

      📌 工作参考价值（4 点）
      • 推荐系统中的 risk-averse 场景（如高价值用户触达、推送时机选择）可借鉴动作条件保底思想：对每个候选动作（推送/不推送）单独构建共形预测集，避免因边际保证导致某些动作下风险失控。
      • 多智体动作选择中，可为每个 agent 的动作输出附带安全区间，防止某个动作因不确定性极高而拖垮整体收益，尤其适合电商广告自动出价等需要逐动作风险控制的决策。
      • 工程实现上 pinball loss 最小化为共形预测提供了一种无需重训练的有限样本算法，可直接集成到现有 ML pipeline 的后处理中，降低落地成本。
      • 在生成式推荐的候选集过滤阶段，用动作条件的 coverage 保证替代全局 top-k 截断，能提升对长尾商品/新商品的探索安全性，平衡收益与风险。

      Conformal Certification of Reasoning Trace Prefixes

      推理轨迹前缀的保形认证

      🧭 推理可靠性 · 保形风险控制 ★ 深度解读

      Rice University

      提出 CROP 方法，为推理步骤序列提供前缀污染风险的有限样本边际控制，保留有效中间推理以提升下游修复

      📌 工作参考价值（4 点）
      • 在多步 Agent 决策中，可用 CROP 对已执行的步骤序列进行风险认证，返回风险可控的“安全前缀”，将剩余步骤交给人工审查或更高精度的修复模块，避免全盘回滚。
      • 生成式推荐（GenRec）在生成物品序列（如购物车推荐）时，可以用类似前缀认证思路，将生成过程按步骤拆分，用 process reward model 标注风险，仅保留低风险前缀再继续生成，提高推荐列表的整体可信度。
      • 工程实现上，CROP 是 verify-agnostic 的轻量校准层，只需将任意步骤级风险评分（PRM、logits、heuristics）输入即可获得保形保证，可直接插入现有的 LLM 推理管线，无需修改模型。
      • 实验揭示了“步骤 AUROC 高 ≠ 前缀实用性好”这一重要现象，可迁移至电商搜索排序中：离线排序指标（如 AUC）可能无法反映线上截断位置的效用，评估应直接面向最终截断策略校准。
