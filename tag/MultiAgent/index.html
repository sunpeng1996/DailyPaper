      InterleaveThinker: Reinforcing Agentic Interleaved Generation

      InterleaveThinker：多智能体增强交错图像生成

      🧭 多智能体交错生成 · 强化学习 ★ 深度解读

      CUHK MMLab · Meituan · CUHK IMIXR

      首个解耦规划与评判的多智能体框架，让任意冻结图像生成器具备强交错生成能力，并解决视觉过依赖与误差累积。

      📌 工作参考价值（4 点）
      • 多步决策解耦范式：面对长程任务时，可借鉴 Planner-Critic 分离设计，让 Planner 一次性生成全局计划以屏蔽中间视觉反馈，避免“过依赖”导致的中途停滞或目标漂移，Critic 专注局部校正。该设计可直接迁移至多轮对话推荐、商品序列生成等场景。
      • 双奖励单步RL降本增效：提出的 accuracy reward（判断准确度）和 step-wise reward（精炼前后质量差）将全轨迹 RL 简化为单步 RL，大幅降低训练成本，适合长工具调用链的 Agent 训练。在电商文案生成、多步商品图编辑等长序列任务中，可仿效此策略设计局部奖励信号，实现高效全局优化。
      • 高质量训练数据构造流水线：通过类别树、实体库和模板生成多样化种子提示，再用强模型生成完整规划-执行-评判轨迹，最后用迭代评分过滤并分裂为 SFT/RL 数据。该方法可用于构建 multi-agent 的规划-反思训练集，尤其适合推荐系统智能体或交互式内容生成场景。
      • 基于评分方差的数据划分策略：将迭代评分方差大的样本专用于 RL，使 Critic 学习精炼能力；方差小的样本用于 SFT 冷启动。这一技巧可直接复用到任何“建议-执行-评价”的强化学习训练数据准备中。

      MLEvolve: A Self-Evolving Framework for Automated Machine Learning Algorithm Discovery

      📌 工作参考价值（4 点）

      Strabo: Declarative Specification and Implementation of Agentic Interaction Protocols

      Strabo：声明式多智体交互协议的声明与实现

      🧭 声明式交互协议与工业标准桥接 ★ 深度解读

      North Carolina State University · Lancaster University

      用声明式协议 Langshaw 精确建模工业电商交互标准 UCP，并通过桥接层实现与现有系统互操作。

      📌 工作参考价值（4 点）
      • 声明式协议减少隐式假设：像 Langshaw 那样显式定义角色、数据所有权（sayso）、动作因果依赖和排斥约束，可让电商多智能体交互（如结算、退货）更清晰，避免因文档不明确导致的实现错误。

      智能体思维链转向：高效可控的 LLM 推理框架

      🧭 LLM 推理控制 · 预算自适应策略转向 ★ 深度解读

      University of California San Diego · Intuit AI Research

      将推理过程建模为马尔可夫决策过程，用可训练的控制器代理在推理过程中动态选择策略和转向短语，实现预算感知的高效思维链转向

      📌 工作参考价值（4 点）
      • 分步导向的推理控制可迁移到多步业务任务：电商场景中的复杂客服、多轮谈判、商品对比推荐等需要长链推理的任务，可以采用类似的控制器-推理器架构，将推理过程分解为可引导的步骤（理解、规划、执行、检查、总结），在每一轮通过轻量级控制器选择最优策略短语，在保持答案质量的同时大幅降低 token 开销。
      • 预算条件奖励塑形设计可控制生成长度与成本：论文提出的非对称奖励函数（正确但超预算降分，错误但未用尽预算加重惩罚）可直接用于推荐系统中的生成式文案或解释生成，通过设定 token 预算实现质量与成本的精细权衡，避免长而不实的输出或过早截断导致错误。
      • 异步控制器-推理器部署架构降低延迟：将控制器与推理器部署为独立的 SGLang 服务器并通过异步 HTTP 通信，可以实现与直接批量推理相仿的吞吐量，这一架构可借鉴到在线推荐系统的多智能体编排中，以低延迟实现复杂的多步决策。
      • 可解释的节省来源分解支持精准优化：论文将 token 节省区分为缩短正确答案、挽救错误尝试、提前终止错误路径等类型，业务系统可借鉴这种分类来定位推理环节的冗余部分，有针对性地优化推理策略，如减少不必要的验证轮次。
