      Holistic Data Scheduler for LLM Pre-training via Multi-Objective Reinforcement Learning

      基于多目标强化学习的LLM预训练全局数据调度器

      🧭 LLM预训练在线数据混合

      China Electronics Technology Group Corporation 15th Research Institute · Renmin University of China · BRAIN · Alibaba Group

      用多目标强化学习动态混合训练数据，兼顾数据质量、域间影响和模型自身信号

      📌 工作参考价值（4 点）
      • 训练电商/搜索/推荐场景的多域统一模型（如用户行为序列模型）时，可借鉴其多目标奖励设计：同时考虑各域数据质量指标、域间损失信号交互（如搜索域的 loss 变化对推荐域的预测影响），以及模型参数范数等内在反馈，用 RL 动态调整各域采样比例，替代固定混合策略。
      • 工程实现上，采用 Soft Actor-Critic (SAC) 算法在连续空间中输出各域采样权重，比离散调度更平滑，且通过离线回放池减少在线交互成本；该框架可直接复用到多数据源的 Embedding 训练或生成式推荐模型的 token 混合训练。
      • 实验显示 HDS 可节省 44% 训练步数达到同等困惑度，且下游任务（MMLU）提升 7.2%，表明更高效的数据利用——在业务资源有限时，用该方法加速大模型训练收敛，避免昂贵的全量等比例训练。
      • 注意该方法需要实时监控多域验证损失，业务落地时须搭建配套的评估流与 RL 控制回路，初期可先在离线训练中模拟不同混合策略的效果，再切换到在线调度。

      Learning Coordinated Preference for Multi-Objective Multi-Agent Reinforcement Learning

      面向多目标多智能体强化学习的协调偏好学习

      🧭 多智能体多目标偏好协调学习 ★ 深度解读

      University of Arizona

      通过让智能体学习协调的个性化偏好向量，实现团队角色分工和帕累托前沿覆盖，提升多目标多智能体协作性能

      📌 工作参考价值（4 点）
      • 多智能体协作系统可借鉴“个性化偏好分工”思路：不让所有 Agent 共享相同的目标权重，而是为每个 Agent 学习专属的偏好向量，促进角色分化（如一个侧重效率，另一个侧重安全），从而在整体目标上获得协同增益。这在推荐系统的多召回通道、多目标排序模型的协同中值得尝试。
      • 偏好的学习可建模为隐变量协调：使用基于 Dirichlet 分布的随机偏好规划器根据局部观测采样偏好，再输入偏好条件策略执行动作；训练时通过团队优势信号和多样性正则项更新规划器，实现端到端的偏好协调。这一机制可迁移到多 Agent 推荐或广告竞价场景，让不同 Agent 自适应地关注不同指标。
      • 偏好多样性正则项（最大化 Agent 间偏好分布的 pairwise 距离）可防止所有 Agent 偏好坍缩至相同方向，强制形成互补的分工；在工程上，这类似于在多个策略网络间引入差异化约束，提升整体系统的鲁棒性。
      • 理论分析表明偏好多样性能够为团队目标带来一阶增量改善，这意味着在设计多目标多智能体系统时，可以优先考虑在偏好空间中增加多样性作为廉价的性能提升手段。

      Not Every Rubric Teaches Equally: Policy-Aware Rubric Rewards for RLVR

      POW3R：基于策略感知的动态评分标准奖励框架

      🧭 RLVR · 动态奖励聚合 ★ 深度解读

      Scale AI · Persona

      提出POW3R框架，通过rollout对比自适应调整评分标准权重，将人类重要性与当前可学性解耦，提升GRPO训练效果与效率

      📌 工作参考价值（4 点）
      • 多目标奖励的动态权重分配：在电商推荐或Agent的多维度评估中（如准确性、多样性、响应速度），可借鉴POW3R，根据策略当前表现动态调整各维度在奖励中的压力，避免固定权重导致学习停滞在已饱和或无法达标的目标上。
      • 信息量驱动的训练信号筛选：利用rollout级别的方差（或评分者间分歧）识别哪些评估项对当前策略最具区分力，将训练压力集中于此，其他项保留基线权重，这能提升早期训练效率和最终性能，类似推荐系统中对困难用户或长尾商品的强化学习倾斜。
      • 保持类别质量平衡的归一化设计：POW3R在类别内重新分配权重但保持类别总质量不变，这种设计可复用至Agent的多能力维度（规划、工具使用、解答准确性）训练，避免某一能力因梯度占优而掩盖其他能力的发展。
      • 评估与训练奖励解耦：静态的人类重要性权重保留作为最终评估目标，训练时则使用动态权重提供更丰富的梯度信号。这种拆解对生成式推荐中多语义ID的优化同样适用：静态的语义重要度不直接等同于RL训练时的有效信号，可分别设计评价和训练两套权重。
