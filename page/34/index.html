      Gated Bidirectional Linear Attention for Generative Retrieval

      用于生成式检索的门控双向线性注意力

      🧭 生成式推荐 · 线性注意力 ★ 深度解读

      Yandex

      提出GBLA混合编码器，以1:2交错自注意力与线性注意力，在长序列生成式检索中匹配双向自注意力质量，推理加速最高8.2倍

      📌 工作参考价值（4 点）
      • 混合编码器是工程落地的优选策略：在生成式推荐编码器中，以[SA, LA, LA]模式（1个标准自注意力层 + 2个线性注意力层）替换全自注意力，能在几乎不损失召回率的同时大幅降低长序列下的延迟，适合历史行为序列极长的电商或流媒体场景。
      • GBLA的三个轻量组件可以直接复用：针对线性注意力添加上下文卷积（Conv1D）、Key门控（soft forgetting）和RMSNorm门控输出，每个组件带来约0.5-1个点的Recall提升，实现成本低，可在自研模型里快速实验。
      • 双向注意力对编码器至关重要：实验表明编码器用双向掩码比因果掩码在Recall@10上绝对高出9个百分点以上，设计生成式推荐编码器时应坚决保留双向上下文。
      • 长序列推理加速的临界点：在H100上，序列长度超过4096时线性注意力开始反超FlashAttention-v3，到32K时加速8.2倍；这提示在用户序列长度上万级别时，线性注意力是延迟瓶颈的有效解法。

      Breaking the Bubble: Asynchronous Pipeline Parallel Training with Bounded Weight Inconsistency

      🧭 分布式训练 · 流水线并行优化

      Technion - Israel Institute of Technology · Ben-Gurion University of the Negev

      通过局部梯度积累作为版本控制，零气泡异步流水线以有界权重漂移换取训练加速

      📌 工作参考价值（4 点）
      • 梯度累积被重新用作一种轻量级版本控制机制：通过增大累积步数减缓参数版本演化，使异步流水线中微批次跨越的优化器更新次数有界，从而控制权重不一致，这个思路可直接用于训练推荐大模型或 Agent 的 LLM 骨干网，无需额外校正模块。

      DNQ: Deep Nash Q-Network for Partially Observable n-Player Games

      DNQ：部分可观测 n 人博弈的深度纳什 Q 网络

      🧭 多智能体博弈 · 求解器监督策略学习

      交替训练框架，利用均衡求解器为多智能体策略提供监督信号，并通过成对分解实现可扩展的博弈训练。

      📌 工作参考价值（4 点）
      • 均衡求解器作为策略监督：对于电商竞价广告、多智能体资源分配等非平稳环境，可直接将外部博弈求解器（如 Nash 均衡计算）生成的目标分布作为策略模仿的监督信号，避免传统 RL 的不稳定训练。
      • 成对分解降低复杂度：当智能体数量多时，将 n 人博弈分解为每对智能体的双人博弈，大幅降低均衡求解成本和训练时间，适合大规模多智能体场景（如多商家竞价）。
      • 掩码策略用于合法动作：在策略模仿时对不合法的动作（如超出预算的竞价）进行掩码，只对合法动作分布计算 KL 散度，确保策略始终满足约束，可直接借鉴到带约束的推荐出价策略中。
      • 共享批评家摊销收益学习：用一个共享网络预测所有智能体对的收益矩阵，减少参数并加速收敛，适用于智能体间存在对称或相似属性的场景，如电商平台多个类似卖家竞价。
