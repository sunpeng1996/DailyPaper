      Institute of Foundation Models (IFM) · Carnegie Mellon University

      分解智能体为反应执行、模拟规划和自调节三系统，用更少推理token达到竞争性准确率。

      📌 工作参考价值（4 点）
      • 在Agent工作流中引入配置器模块：可借鉴SR²AM的配置器（System III）设计，对每一步状态评估是否需要调用规划模块（System II），避免不必要的深度推理。在电商对话或推荐Agent中，可根据用户问题复杂度、当前信息充分度动态决定是否进行多步推理，从而降低平均推理成本。

      • 数据构造方法：从强模型轨迹重建结构化规划：v1.0通过注释器从已有链式思考中提取配置器决策和模拟规划内容，再用于监督微调，可作为一种高效的数据增强策略。如果业务中已有成功Agent的完整对话轨迹，可以用类似方法逆向构造结构化SFT数据，快速冷启具有自调节规划能力的小模型。

      Bridging the Cold-Start Gap: LLM-Powered Synthetic Data Generation for Natural Language Search at Airbnb

      用 LLM 合成查询与标签破解自然语言搜索冷启动

      🧭 自然语言搜索 · 合成数据冷启动 ★ 深度解读

      Airbnb

      用少量种子查询引导 LLM 生成带标咨询数据和标签，使查询分布接近真实用户，支持检索与排序模型冷启动训练

      📌 工作参考价值（4 点）
      • 用种子查询控制 LLM 生成风格：直接让 LLM 无约束生成会产出冗长、多属性的查询，与真实用户短促、不规范的表达严重不匹配。从用户调研或早期日志中收集数百条种子查询作为模板，可以大幅降低 KL 散度（本文从 4.95→0.66），实现更逼真的查询分布。在电商搜索、对话式推荐中可复用此思路，用小样本种子让合成数据贴合真实语言。
      • 三类提示模板平衡真实性与多样性：seed_controlled 严格保持种子结构，适合属性覆盖；seed_freeform 模仿风格但允许变化；variety 放松约束、提高温度，探索长尾意图。实际生成时按比例混合三种模板，既能保证主流查询的真实感，又能覆盖边缘场景。推荐系统在生成候选集时可借鉴这种分阶段控制的提示策略。
      • 困难样本构建的分层采样与提示技巧：为避免全是简单样本（所有模型都能答对），通过限制样本对相似度（同类别、嵌入距离过滤）和提示中加入“难度桶”、反事实编辑等手段生成有区分度的训练数据。这对于提高 ranking 模型的迭代信号非常重要，电商场景下可借鉴硬负样本挖掘来提升模型判别力。
      • 从 LLM 自生成数据中避开自偏好陷阱：用 LLM 评估 LLM 生成的内容容易出现系统性偏爱（本文 GPT 评估自身生成的对准确率高达 99%），实际训练中建议用构建式标签（对比生成）保证高保真，用 LLM 评估仅作离线质量检查；且通过属性锚定、人工校准、使用异构模型降低自我偏好风险。这对任何用 LLM 生成训练数据的团队都是重要提醒。

      Layer-wise Token Compression for Efficient Document Reranking

      逐层令牌压缩以提升文档重排效率

      🧭 文档重排序效率优化 · Token 压缩 ★ 深度解读

      Amazon AGI · Amazon AWS

      在中层Transformer层进行自适应令牌池化，保留早期交互并加速计算，吞吐量最高提升116%

      📌 工作参考价值（4 点）
      • 电商搜索/推荐的重排序阶段常使用 cross-encoder，文档/商品描述较长导致推理延迟高；可借鉴 LTC 在中间层做 token 池化，根据吞吐需求调节压缩率与层位置，平衡效果与速度。

      • 轻量压缩（如保留 80% tokens）可能产生正则化作用，使得模型在未见过的长文本上泛化更好，可用来处理商品描述长度差异大的情况，无需额外训练。

      • 多模型协作打破自反馈循环：使用一个结构不同的peer模型（如主模型用SASRec，peer用Mamba4Rec）提供外部负样本打分，可以注入异质信号，防止模型自引导退化。电商序列推荐可在线上训练时同时维护两个异构模型，互相提供负样本候选。

      • 即插即用的通用框架：方法对MLP、RNN、Transformer、Mamba等主流序列模型均有效，且同时适配BCE和BPR损失。在已有训练流程中引入peer模型和teacher蒸馏头即可，部署成本低，可直接作为负采样模块嵌入现有电商推荐训练管线。
